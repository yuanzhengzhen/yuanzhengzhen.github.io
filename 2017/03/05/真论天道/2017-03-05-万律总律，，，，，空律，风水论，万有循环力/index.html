小学性读本引吐槽:小学生性教育读本完整版是黄色漫画
来源:澎湃新闻
2017-03-04 09:06
http://www.mnw.cn/ 海峡都市报电子版

　　网友评论

　这套名为《珍爱生命—小学生性健康教育读本》的教材自2010年开始由北京师范大学出版社陆续出版，主编系北京师范大学儿童性教育课题组负责人刘文利。
　　事实上，早在2015年，知乎上就有一名职业是小学教师的网友（知乎ID：心诚，链接：https://www.zhihu.com/question/33593693/answer/58999207）发表过对这套教材的使用心得，评价是“很不错”，“视野开阔，角度中肯”，“从一年级到五年级每学期一本，内容涉及学龄儿童情绪管理、社交、性健康、性别权利等诸多方面”。
　 　　这位教师表示，尽管也有“内容这么直白，难以启齿”的时候，有些同事也“犹豫、拖拉、不敢讲”，但是因为北师大的儿童性教育课题组给老师们精心准备过几次培训，“教我们怎样坦诚地对学生讲课，不要羞于启齿，还教我们怎样安排课程，设计教案。”他表示：“关键要坦诚，不能表现出羞于启齿的心理。”

　　关于学生对这套教材的反馈，这位教师也有生动的描述：
　　“孩子们刚拿到书，打开一看，好多孩子都捂住眼睛，大喊，啊，羞羞脸！啊，我不敢看！……当然，也有一些孩子是很坦然地看的，因为他们心里没有那种以性为羞耻的观念。当我开口讲课时，有个别孩子居然盖上书本，捂住耳朵，说，我不听，我不听。我就严肃地对学生说，这些知识都是讲我们自己的身体，每个人都有，每个人都会发生，我们为什么不了解一下它们呢？”
　　“三年级学生基本没有性欲望，都当普通知识学了。当然，他们是很好奇的，于是争着传阅二年级的书（讲人的诞生）。”
　 　　澎湃新闻记者在北京师范大学儿童性教育课题组的网站（http://www.sexualhealth-education.org）上看到，课题组“以‘将性教育纳入中国义务教育课程体系’为己任”，以北京市流动儿童学校为实验学校，目前已在北京市十余所流动儿童学校开设了系统的小学性教育课程。而刘文利博士毕业于美国内布拉斯加大学，现就职于北京师范大学脑与认知科学研究院，从1988年开始从事儿童性教育研究和实践工作至今。

博主贴在网上的教材，但只拍到画面的一半，原意为“不要给别人看自己的生殖器”。有网友认为不断章取义的话就没问题。　

　　据该网站介绍，课题组是“依据教育部《中小学健康教育指导纲要》，并将联合国教科文组织《国际性教育技术指导纲要》本土化，研发了小学1～6年级的性教育课程内容，每个年级12课时，六年共72课时”。“这套教材的问世，使中国成为世界范围内第一个将《国际性教育技术指导纲要》本土化，并出版小学全套性教育教材的国家。”
